# Product Overview

Amazon Connect MCP Server is a Model Context Protocol (MCP) server that enables AI assistants to interact with Amazon Connect contact centers. It provides 86 tools covering all major Amazon Connect services including cases, contacts, configuration, analytics, profiles, campaigns, and AI features.

## Key Features

- **Multi-Region Support**: List and manage instances across all AWS regions
- **Configuration Templates**: Pre-built templates for cases, views, routing, and infrastructure
- **Setup Wizard**: Guided setup for new Connect instances with Infrastructure as Code generation
- **Tiered Architecture**: Core tools (Tier 1) for essential operations, domain-specific tools (Tier 2) for specialized tasks

## Target Users

- Contact center administrators and developers
- AI assistants working with Amazon Connect
- DevOps teams managing contact center infrastructure
- Support teams needing programmatic access to Connect services

## Integration

Designed to work with MCP-compatible AI assistants including Claude Desktop, Cursor, and Kiro CLI through the FastMCP framework.
